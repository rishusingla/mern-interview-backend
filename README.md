# MERN Interview Backend

This is the backend API for the MERN Interview Platform. It handles authentication, user management, and interview data.


🔧 **Live API:** [Click Here](https://mern-interview-backend-jmpz.onrender.com)
🌐 **Frontend Repo:** [Click Here](https://github.com/rishusingla/mern-interview-frontend/blob/main/README.md)

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB (Atlas)
- JWT Authentication
- Mongoose

## 📦 Installation
git clone https://github.com/your-username/backend-repo.git
cd backend-repo
npm install
npm run dev
